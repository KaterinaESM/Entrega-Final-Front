🛒 E-commerce Talento Tech

Sitio web de e-commerce dinámico e interactivo desarrollado como proyecto final del curso Talento Tech.
La aplicación permite visualizar productos obtenidos desde una API REST, agregarlos a un carrito de compras y gestionar la compra mediante una interfaz responsive y accesible.

🚀 Tech Stack
HTML5 (estructura semántica)
CSS3
Flexbox
CSS Grid
Bootstrap
JavaScript
Fetch API
LocalStorage
Formspree
Google Fonts
